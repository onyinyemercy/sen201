# Student Registration Form System

## Project Description

The Student Registration Form System is a simple web-based application designed to collect and validate student information using HTML, CSS, and JavaScript. The system performs client-side validation and provides user feedback without requiring a backend server.

## Features

- Student data collection
- Client-side validation
- Clean and responsive UI
- Single-file implementation

## Technologies Used

- HTML
- CSS
- JavaScript

## Form Fields

- First Name
- Middle Name
- Last Name
- Matric Number
- Department
- Level
- Email Address
- Phone Number
- Gender
- Date of Birth
- Home Address

## How to Run

1. Download or clone the repository
2. Open `index.html` in any modern browser
